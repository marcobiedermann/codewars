# [Sum of all arguments](https://www.codewars.com/kata/540c33513b6532cd58000259)

Write a function that finds the sum of all its arguments.

eg:

```javascript
sum(1, 2, 3); // => 6
sum(8, 2); // => 10
sum(1, 2, 3, 4, 5); // => 15
```

```php
sum(1, 2, 3) // => 6
sum(8, 2) // => 10
sum(1, 2, 3, 4, 5) // => 15
```

```python
sum_args(1, 2, 3) # => 6
sum_args(8, 2) # => 10
sum_args(1, 2, 3, 4, 5) # => 15
```

```coffeescript
sum 1, 2, 3 # => 6
sum 8, 2 # => 10
sum 1, 2, 3, 4, 5 # => 15
```

```ruby
sum 1, 2, 3 # => 6
sum 8, 2 # => 10
sum 1, 2, 3, 4, 5 # => 15
```

```clojure
(sum 1 2 3) ; => 6
(sum 8 2) ; => 10
(sum 1 2 3 4 5) ; => 15
```

```c
sum(3, 10, 20, 3); // --> 33
sum(2, 10, 15); // --> 25
sum(1, 10); // --> 10
sum(0); // --> 0
```
